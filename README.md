# Academic Performance Tracker

A modern GPA/CGPA tracker built with Next.js for students to calculate, organize, and plan their academic performance.

## Features

- GPA & CGPA calculation
- Subject-wise grade management
- Manual GPA entry
- Semester-wise tracking
- Responsive UI
- Dark theme
- Local data persistence

## Tech Stack

- Next.js
- TypeScript
- CSS Modules

## Live Demo

[here](https://academic-performance-tracker-pi.vercel.app/)

## Run Locally

```bash
git clone https://github.com/dauxl/Academic-Performance-Tracker.git
cd Academic-Performance-Tracker
npm install
npm run dev
```

Open `http://localhost:3000`.
